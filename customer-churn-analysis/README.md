# Telco Customer Churn Analysis

This project analyzes customer churn for a telecommunications company using
Python, MySQL, SQL, and Tableau tools. It demonstrates an end-to-end data analyst
workflow from raw data ingestion to executive reporting.



### Dataset
IBM Telco Customer Churn dataset containing customer demographics, services,
billing information, and churn status.

<br/>


### Tools & Technologies
- Python (Pandas, NumPy)
- MySQL
- SQLAlchemy
- SQL
- Tableau
- Jupyter Notebook

<br/>

### How to Run
1. Start MySQL server
2. Create database `analytics_db`
3. Configure `.env`
4. Run:
    ```bash
    python scripts/main.py
    ```
